# Лабораторная работа №3 по курсу "Операционные системы"

## Описание
Программа для фильтрации строк с использованием межпроцессного взаимодействия через Shared Memory.

**Вариант:** 18 (Лаб 1)

## Функциональность
- Родительский процесс читает входные данные
- **Нечетные строки** отправляются в `client1`
- **Четные строки** отправляются в `client2`
- Дочерние процессы удаляют все гласные буквы из полученных строк

## Компиляция

```bash
gcc -o server server.c -lrt -lpthread
gcc -o client client.c -lrt -lpthread
```

## Запуск 

```bash
./server
```

## Порядок ввода

- **Первые две строки** - названия файлов, для первого и второго дочерних процессов соответственно
- **Далее** текст для обработки строго на английском
- **Для завершения ввода** ввести EOF
